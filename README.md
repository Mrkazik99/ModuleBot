Simple modular bot for telegram. You can add any module adhoc (For now you need to restart bot to make new module working, but I'm trying to resolve this problem xD)

To show inline answer return dicts in this form:

{'<label>': {"text": "<output-text>", "desc", "<output-description>", "withImage": "<False>"}

OR

{'<label>': {"text": "<output-text>", "desc", "<output-description>", "withImage": "<True>", "imageType": "<image-type>", "imageUrl": "<image-url>"}
